### 功能介绍

- 登陆
- 首页
- 每日推荐
- 歌单广场
- 排行榜
- 云村热评
- 视频
- MV详情页
- 我的
- 电台模块【电台首页，电台详情, 电台排行榜】
- 搜索【支持单曲，MV，专辑，歌单，电台】
- 播放页【歌词，播放列表，上一首，下一首】
